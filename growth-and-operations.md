# Рост и операции — @aleshka_reallife

> Закрывает оставшиеся пробелы аудита: механики роста (кроме «постить»), лучшее время постинга
> (на реальных данных), пер-ролик трекинг форматов, платный разгон и kill-switch/пивот.

---

## 1. Механики роста, кроме постинга

Постинг — половина роста. Вторая половина — активные петли:

- **Комменть крупные аккаунты ниши** (Кожемяко, rezedkin и др.): первым, смешно, в тему. Их аудитория
  видит тебя → часть переходит. Дешёвый органический приток «на чужом охвате».
- **Коллабы/совместные ролики** с блогерами твоего размера (2–10k): «дуэт абсурда», обмен упоминаниями.
  Взаимный пиар = обмен аудиториями без бюджета.
- **Реагируй на ролики других через Reels** (со своим слоем — см. `risk-and-originality.md`): отмечаешь
  автора, он может репостнуть → его аудитория идёт к тебе.
- **Отвечай видео-роликом на комменты** — формат «ответ на коммент» алгоритм любит, а комментатор приводит своих.
- **Кросс-промо с Telegram и TikTok** — гоняй аудиторию между своими площадками (см. `distribution-multiplatform.md`).

Правило: **1 активное действие в день** помимо постинга (коммент/коллаб/ответ). За месяц — 30 точек входа.

## 2. Лучшее время постинга (на твоих реальных данных) 📊

Разбор публикаций за 90 дней (время МСК = UTC+3; аудитория — СНГ, несколько часовых поясов):

**Что показали данные:**
- **Крупнейшие охваты — вечер МСК 20:00–23:00.** Вирусный ролик вышел в пн **23:00 МСК**; сильные
  органические всплески (охват 3 400 и 4 000) — в пт **21:00–22:00 МСК**.
- **Вторичное окно — утро МСК 10:00–12:00** (апрельские ролики на 1 800 охвата).
- **Слабое время — выходные утром/днём** (охваты 350–600).
- **Лучшие дни — понедельник и пятница** (на них пришлись все крупные всплески).

**Рекомендация:**
| Слот | Время (МСК) | Для чего |
|---|---|---|
| 🥇 Основной | **20:00–22:00** | ежедневный ролик, флагман (формат E) |
| 🥈 Тест | **22:00–23:00** | поздний вечер — на нём стрельнул вирус |
| 🥉 Второй за день | **10:00–12:00** | второй ролик в дни двойного постинга |
| Флагман (E) | **понедельник / пятница вечер** | лучшие исторические дни |

⚠️ Оговорка: охват сильнее зависит от **качества ролика**, чем от часа — время лишь докручивает.
Первые 2 недели протестируй эти слоты и уточни по своим Insights (это же окно проверит `weekly-report-auto.md`).

## 3. Пер-ролик трекинг форматов (чтобы знать, что реально работает)

Отчёт даёт общие цифры, но не говорит, **какой формат конвертит**. Веди простую таблицу (Google Sheets/заметки):

| Дата | Формат (A/B/C/D/E) | Тема | Охват | Просмотры | Репосты | Сохран. | Комменты | Новые подписч. | Реп/охват % |
|---|---|---|---|---|---|---|---|---|---|
| | | | | | | | | | |

- Заполняй раз в неделю из Insights (или попроси меня — вытяну из Windsor.ai).
- Раз в месяц смотри: **у какого формата выше «репосты/охват» и «подписки/охват»** — его и масштабируй,
  слабый убирай. Это превращает контент-план из гадания в управление по данным.

## 4. Платный разгон (Meta Ads подключён) 💸

Органика — основа, но маленький буст усиливает уже доказанное:

- **Что бустить:** только ролик, который **уже органически залетел** (охват ×3 к обычному). Не лей деньги
  в непроверенное — сначала органика голосует, потом усиливаешь победителя.
- **Бюджет теста:** 300–500₽ на ролик, цель «охват»/«вовлечение», аудитория — мужчины 18–44, РФ/СНГ.
- **Зачем:** добавить топлива в волну и притянуть новых на лучший контент. Это ускоритель, не замена росту.
- **Не бустить:** ради «красивых цифр» или на холодный/сырой ролik — слив бюджета.

Через твой Meta Ads (Windsor.ai) можно поднять бюджет/включить кампанию — но только после твоего подтверждения.

## 5. Kill-switch и пивот: когда признать, что не зашло 🔀

Чтобы не крутить впустую месяцами, зафиксируй критерии проверки через **30 дней**:

| Сигнал через 30 дней | Вывод | Действие |
|---|---|---|
| Подписки на 1000 охвата **> 3** и растут | формат работает | масштабируем, наращиваем оригинал |
| Репосты к охвату держатся **> 1%** | контент «пересылаемый» | продолжаем |
| Подписки/1000 **< 2** при норм. охвате | цепляет, но не конвертит | усилить закреп, CTA, узнаваемость |
| Охват и подписки падают 3 недели подряд | ниша/подача не идёт | менять угол (напр. больше оригинальных скетчей a-la Кожемяко) |

**Человеческий фактор (честно):** мы увели тебя из «семья/личное» в абсурд по данным. Проверь на себе:
комфортно ли делать это **месяцами**? Если да — гони. Если выгораешь — миксуй с личным (формат D),
но не возвращайся в чистый лайфстайл-влог: рынок его уже отверг. Устойчивость автора важнее любой стратегии —
выгоревший канал не растёт.

## 6. Операционный ритм (чтобы не сгореть)
- **1 съёмочный день/нед** (батч) → 7–10 роликов впрок (см. `production-bible-aleshka.md`).
- **Ежедневно:** публикация в слот + первый час вовлечения + 1 активное действие роста (п.1).
- **Еженедельно (пятница):** сверка `weekly-report-auto.md` + teardown 1 конкурента + план на неделю.
- **Ежемесячно:** разбор трекинга форматов (п.3), проверка kill-switch (п.5), обновление медиакита.
